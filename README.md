<h1>Web-Development-Practice</h1>

<p>
    This project is a simple website created to practice basic HTML and CSS. 
    It represents a college game society, including introductory information, events, 
    and general content related to gaming activities on campus.
</p>

<h2>Project Overview</h2>
<p>
    The goal of this project is to develop and improve foundational web development skills.
    The focus is on writing clean, semantic HTML and applying CSS styling effectively.
</p>

<h2>Features</h2>
<ul>
    <li>Homepage introducing the College Game Society</li>
    <li>Sections describing events and activities</li>
    <li>Clean layout built with HTML and CSS</li>
    <li>Beginner-friendly structure</li>
</ul>

<h2>File Structure</h2>

│

    └── styles.css
</pre>

<h2>Technologies Used</h2>
<ul>
    <li>HTML5</li>
    <li>CSS3</li>
</ul>

<h2>Purpose</h2>
<p>
    This project is meant for learning and experimentation. 
    It is not a full production website but a practice environment to test layout ideas, 
    spacing, colour schemes, and general front-end structure.
</p>

<h2>Future Improvements</h2>
<ul>
    <li>Adding responsive design for mobile devices</li>
    <li>Adding JavaScript for interactive features</li>
    <li>Improving layout and styling</li>
    <li>Creating a gallery of game society events</li>
</ul>
